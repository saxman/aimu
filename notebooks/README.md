# AIMU Notebooks

📖 **[View these notebooks rendered as a site →](https://saxman.github.io/aimu/notebooks/)**

GitHub displays `.qmd` files as plain source, so browse the rendered HTML at the link above.

These are runnable tutorial demos, one per subsystem, numbered `01`–`26` to build up
incrementally. They are authored as plain-text [Quarto](https://quarto.org) `.qmd` files
(markdown with executable `python` cells).

**Run them yourself** (see the [main README](../README.md#notebooks) for details and dependencies):

- `quarto preview notebooks/`: a live-reloading local site.
- VS Code with the Quarto extension: run cells in the Jupyter Interactive Window.
- `python notebooks/convert.py to-ipynb`: convert to Jupyter notebooks for inline execution.
